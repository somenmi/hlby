## ✨ Особенности
- 🔗 Сохранение ссылок
- ✍️ Автоматические названия страниц
- 🤓 Иконки сайтов
- 📅 Дата и время добавления
- 🌓 Тёмная/светлая темы
- 💾 Локальное сохранение

## 🛠 Использование
1. Вставьте ссылку в поле
2. Нажмите Enter или зелёную кнопку
3. Клик по ссылке - откроется в новой вкладке
4. Красная кнопка урны - удалить

## 💾 Хранение данных
Все данные сохраняются в `localStorage` браузера

    {
        id: "17***-abc123",
        url: "https://somenmi.github.io/hlby/",
        title: "Название страницы",
        added: "2000-02-02T02:02:00.000Z"
    }
